# Content checklist before publishing

Replace every placeholder before you publish the site publicly.

## Required replacements

- `Your Name`
- `yourname.com`
- `your.email@example.com`
- `your.email [at] example [dot] com`
- `YOUR_GITHUB_USERNAME`
- `YOUR_GOOGLE_SCHOLAR_ID`
- Paper titles, coauthors, journal information, DOI links, arXiv links
- CV file at `cv/cv.pdf`

## Optional items

- Add `assets/portrait.jpg` and uncomment the portrait line in `index.html`.
- Delete `problems.html` if you do not want an open problems page.
- Delete `teaching.html` if you do not want a teaching page.
- Add `sitemap.xml` after replacing the example URLs in `sitemap.xml.example`.
- Add a real `CNAME` file only after buying and configuring a custom domain.

## Search visibility tips

- Use the same English name everywhere: homepage, CV, Google Scholar, arXiv, ORCID, GitHub.
- Put your homepage link on your Google Scholar profile, arXiv author page if possible, ORCID, GitHub profile, and school page.
- Keep the homepage URL stable.
- Avoid Google Fonts, YouTube embeds, Google Analytics, and other blocked or slow third-party resources if China accessibility matters.
